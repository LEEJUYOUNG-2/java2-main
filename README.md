### 감지 컬렉션, 메모리 변환
감지컬렉션 = 사용하지 않는 메모리를 자동으로 정리하는 기능 = 반환 명령없이 메모리를 잡아먹지 않도록 함
메모리 변환 = 잠심 멈춤으로써 이뤄지면 실시간 응용 프로그램에 부적합
